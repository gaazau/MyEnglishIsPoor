# My English is poor
一款支持记录英语单词本的小软件

## 功能概览
- 支持从文章中提取英文单词和短语，生成单词本
- 支持标记单词状态:未读,标记,掌握和停用四种状态
- 支持将单词本上传至不背单词app(需在config.json填写用户名和密码)
